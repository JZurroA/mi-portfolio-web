# Mi Portfolio Web

## Integración de Git y GitHub sin asistencia de IA

Este proyecto es la primera vez que realizo de forma autónoma todo el flujo de trabajo con Git y GitHub, incluyendo la ejecución de los comandos necesarios y, sobre todo, entendiendo lo que estoy haciendo, sin apoyarme en herramientas de IA.

En ocasiones anteriores solía recurrir a la IA cuando me surgían dudas con algún paso o comando concreto.  
En este caso, el único uso de IA ha sido la consulta sobre cómo dar formato al archivo README.md (títulos en Markdown).

Lo que se ve en el repositorio es el resultado de haber seguido las instrucciones de la tarea evaluativa, repitiendo el proceso varias veces hasta comprenderlo correctamente.
